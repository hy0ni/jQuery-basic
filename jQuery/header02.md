$(document).ready(function(){

  // 스크롤바가 내려가면 header태그에 활성화 스타일을 넣어주는 코드

  $(window).scroll(function(){
    var sTop = $(window).scrollTop();
    // console.log(sTop);
    if(sTop>=100){
      $("header").addClass("on");
      $(".ball").addClass("rotate");
      var red = Math.random()*255; //0~1까지의 랜덤한 숫자를 생성
      var green = Math.random()*255;
      var blue = Math.random()*255;
      $("header").css("background","rgb("+red+","+green+","+blue+")")
    }else{
      $("header").removeClass("on");
      $(".ball").removeClass("rotate");
      var red = Math.random()*70+50; //50~120까지의 랜덤한 숫자를 생성
      var green = Math.random()*255;
      var blue = Math.random()*255;
      $("header").css("background","rgb("+red+","+green+","+blue+")")
    }
